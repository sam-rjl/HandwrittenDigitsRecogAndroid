# HandwrittenDigitsRecogAndroid
大学生创新创业项目--基于安卓的手写体数字识别

模型搭建用的keras CNN,数据集为传统的mnist数据集
用OpenCV 进行图像分割
最终完成的功能可以进行多书写体数字实时识别
